# Folio: A Portfolio Management Platform

## ⚛️ Tech Stack: 
MongoDB, Express, React.js, Node.js, Redux, and Tailwind

## Setup .env file:
```
MONGO_URI=...
PORT=...
PORTFOLIO_URL=...
DASHBOARD_URL=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
```

## Build the app:
```
npm install
npm run build
```
in the respective folders

## Start the app:
```
npm run dev
```
in the respective folders
